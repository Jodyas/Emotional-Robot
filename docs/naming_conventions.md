# Naming Conventions

- **Variables and Parameters**: `snake_case` using descriptive full words. Avoid abbreviations like `ctx`, `usr` in favor of `context`, `user`.
- **Functions and Methods**: `snake_case`, utilizing verb-noun phrasing indicating action.
- **Classes**: `PascalCase`.
- **Constants/Configs**: `UPPER_SNAKE_CASE` typically located in the `config` layer.
